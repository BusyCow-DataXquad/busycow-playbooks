# Inventory Management — Database IDs

# To find a database ID: open the database in Notion, copy the URL.

# Example URL: https://www.notion.so/myworkspace/abc123...def456?v=...
# Example ID: abc123def456... (formatted as xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx)

# Product Catalog — single source of truth for all stock numbers.

# Dealer List — master record of all reseller/dealer partners.

# Outbound Records — every shipment logged here.
